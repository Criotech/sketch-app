# Sketch APP

A simple tool collaborative sketch / collaborative white board 

# Features


- Client: Next JS/typescript.
- Backend: Express/Typescript, mongodb, socket-io

## Running

Clone this repo, and cd into it, Then to run client :

### `yarn install`
### `yarn run dev`

Clone this repo, and cd into it, Then to run client :

### `tsc`
### `nodemon start`
